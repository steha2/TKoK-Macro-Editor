변수 선언 #...#, 사용 %...%                                #limit:0#

#키:값# ← 값을 계산한 후 키 변수에 할당
예:  
#x:1+2# → x := 3
#if:x>1# → true

#키=값# ← 값 그대로 키 변수에 할당
예: #msg=hello#

변수는 %변수명% 으로 사용
#a=3# #b=2#
예: chat: %a+b%  ; 5

a가 1보다 클 때만 이 줄을 실행합니다 (조건은 한 줄에만 적용)
#if:a>1# Chat: hello  

조건을 여러 줄 대상으로 바꿉니다
#if_mode=block#

#if:a>1#              ; a가 1보다 클 때 아래 여러 줄을 실행
Chat: hello
Click:L 0.100, 0.200
#end_if#              ; 조건 종료

Read: c_map\classic\map
해당 경로의 key=value 로 저장된 변수값을 읽어옴

#force#
해당 줄을 조건문, 스킵 여부을 무시하고 실행

#break#,
해당 매크로 중단

#limit:10#
명령 10회 실행 후 종료 (#limit# 매크로 전체 중단)

#limit_mode=line#
반복 횟수 대신 줄 수를 기준으로 실행 제한을 적용

#dp_mode=round2#     =ceil  =floor
소수점 아래 2자리 반올림    올림    버림

#coord_mode=screen# 
좌표모드:전체화면 (기본값:실행창)

#start_line=10#
10번 줄 부터 명령문을 실행 합니다.

#skip_mode=vars#
스킵 할때 변수 설정과 명령문을 모두 건너뜁니다.

#send_mode=C#
키 입력, 클릭 명령을 비활성 전송합니다.
일부 프로그램에서는 호환성 문제로 작동하지 않을 수 있습니다.

#send_mode=R#
Chat: 채팅에 문자를 직접 타이핑하듯 입력합니다 (기본은 복붙 방식입니다)
